## How to use:

You need:

- [Node.js 16+ (recommend: 16.19.0 LTS)](https://nodejs.org/en/)
- [Git](https://git-scm.com/book/en/v2/Getting-Started-Installing-Git)

1. Install dependencies:

```bash
npm install libma
# or
yarn add libma
```

2. Run:

```bash
npx libma
```
